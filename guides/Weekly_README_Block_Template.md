# Weekly README Block Template

Purpose: copy-paste block for future weekly README files so class-planning intent is
visible at the week level.

## Copy-paste block

```md
## Lesson Planning Notes

- Keep the week on a hard 60-minute class clock with single-start flow.
- Preserve stimulus with load and volume changes before changing movement patterns.
- Keep warm-ups implement-light and move workout-load rehearsal into Movement Prep.
- Solve bottlenecks before class starts, especially on team and station-heavy days.
- Use built-in rest windows for recovery and reset, not extra teaching drift.
```

## How to use it

- Put this block after the weekly overview and before Daily Workouts.
- Adjust one or two bullets if the week has a special constraint such as missing
  source material, unusual equipment limits, or a heavy gymnastics focus.
- Keep the wording short and operational.
- Do not turn it into a long coach essay.

## Optional add-ons

- Add a note about incomplete source material when a board image is missing details.
- Add a note about protected lanes when multiple team workouts share machines,
  bars, boxes, or wall space.
- Add a note about scaling bias when the week includes technical overhead or rope
  work.

## Example

```md
## Lesson Planning Notes

- Keep the week on a hard 60-minute class clock with single-start flow.
- Preserve stimulus with load and volume changes before changing movement patterns.
- Keep warm-ups implement-light and move barbell rehearsal into Movement Prep.
- Solve bottlenecks before class starts, especially on Wed and Fri team pieces.
- Protect built-in rest so athletes reset before the next interval starts.
```
