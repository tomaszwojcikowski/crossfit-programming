# Wednesday (06/05/2026)

**Setup**
- Equipment: Calorie machine (bike/rower/ski), wall balls 10/6 kg, open running lane, floor line for broad jumps
- Timing: 60 min hard cap
- Stimulus: Long team aerobic chipper. Lap runs and machine calories anchor the structure; wall balls finish the piece under fatigue—manage the pacing early to protect the back half.
- Scaling A: Wall ball to 8/5 kg. Lap runs shortened to shuttle runs if space is limited. Machine calories to 80.
- Scaling B: Wall ball to 6/4 kg. Lap run replaced by 200m shuttle or jog. Machine calories to 60. Burpee broad jumps to burpee step-jumps.
- Logistics: One machine per team. Assign machine and running lanes before class. Teams alternate on solo laps; both run on the marked "together" laps.
- Top Cues: "Find a rhythm, don't sprint." "Steady splits on the machine." "Jump and land soft."

## Warmup (10 min)
- **R1 RAISE (2:00 easy):** Easy machine flush or jog
  - 8 x Leg swings/side
  - 8 x Bootstrappers
- **R2 ACTIVATE (1:30 moderate):** Jog with direction changes or shuttle build-up
  - 6 x Alternating reverse lunges
  - 6 x Broad jumps at low intensity
- **R3 POTENTIATE (1:00 hard):** 2 x acceleration runs or short machine surge
  - 4 x Wall ball squats
  - 3 x Burpee broad jumps (easy landing practice)
- Rest 30s between rounds

## Post-Warmup Mobility (90s)
- Hip flexor stretch: 45s/side
- Pike hamstring stretch: 45s

## Movement Prep (3 min)
- 3 x Burpee broad jumps over a line
- 8 calories on the machine @ workout pace
- 4 x Wall ball shots @ workout load

## Main Conditioning: Team of 2, For Time

- 4 Lap Run (1-1-1-1, alternating laps)
- 100 Calorie Machine (split any way)
- 1 Lap Run (together)
- 10 Line Burpee Broad Jumps (split any way)
- 1 Lap Run (together)
- 100 Calorie Machine (split any way)
- 4 Lap Run (1-1-1-1, alternating laps)
- 100 Wall Ball Shots 10/6 kg (split any way)

**Time cap: 30 min (expected: 22-27 min)**

*Line burpee broad jump: perform a full burpee, then broad-jump over a line on the floor; that's 1 rep.*

## Cooldown & Debrief (3 min)
1. 60-90s easy walk, nasal breathing
2. Couch stretch: 45s/side
3. Figure-4 glute stretch: 45s/side
4. Bright spot: "We managed the machine splits so the back half felt as controlled as the first."
