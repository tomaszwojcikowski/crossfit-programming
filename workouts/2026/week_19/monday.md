# Monday (04/05/2026)

**Setup**
- Equipment: Rack, barbell, plates, box (60/50 cm), AbMat or mat
- Timing: 60 min hard cap
- Stimulus: Back squat peak loading first, then alternating 1-minute intervals—win the burpee/clean buy-in fast enough to score sit-ups and front squats at the end of each minute.
- Scaling A: Reduce squat load by 5%. Burpee box jumps to 50/40 cm. Hang cleans to 40/25 kg. Sit-ups and front squats as written.
- Scaling B: Reduce squat loading by 10-15% throughout. Step-up box jumps. Hang power cleans at 35/20 kg. Tuck crunches instead of sit-ups.
- Logistics: One rack per athlete. Box and AbMat staged at lane during strength so the transition into intervals is under 30s.
- Top Cues: "Drive the floor away." "Explode off the box." "Pull the bar to the hip crease."

## Warmup (10 min)
- **R1 RAISE (2:00 easy):** Easy bike, row, ski, or jog
  - 8 x Glute bridges
  - 8 x Air squats
- **R2 ACTIVATE (1:30 moderate):** Line hops or quick-step box taps
  - 6 x Box step-ups
  - 6 x Front rack elbow lifts with empty bar
- **R3 POTENTIATE (1:00 hard):** 2 x 15s fast box jump rhythm
  - 3 x Jump squats
  - 3 x Empty-bar hang power clean
- Rest 30s between rounds

## Post-Warmup Mobility (90s)
- Couch stretch: 45s/side
- Front rack stretch on rig: 45s/side

## Strength: Back Squats (14 min)
**Every 2:00 for 7 sets:**

- Set 1-2: 5 reps @ 85%
- Set 3-4: 3 reps @ 90%
- Set 5-6: 2 reps @ 95%
- Set 7: 1 rep @ 96% + 2 kg

## Reset Window (3 min)
- Strip the bar to interval loads. Stage the box and AbMat beside the rack. Confirm hang clean weight.

## Movement Prep (3 min)
- 2 x Burpee box jumps
- 4 x Sit-ups
- 3 x Hang cleans @ interval load
- 3 x Front squats @ interval load

## Main Conditioning: Intervals
**8 x 1:00 ON / 1:00 OFF, alternating A and B (4 rounds each):**

- **a.** 8 Burpee Box Jumps (60/50 cm), then max Sit-ups until end of time
- **b.** 8 Hang Cleans (50/30 kg), then max Front Squats until end of time

Score = total sit-ups + total front squats across all 8 work minutes.

**Target:** clear the buy-in in ~40-50s and accumulate 4-8 reps of the scoring movement per interval.

## Cooldown & Debrief (3 min)
1. 60-90s easy walk or spin, nasal breathing
2. Pigeon stretch: 45s/side
3. Bright spot: "I hit every squat number and kept the buy-in fast enough to score each minute."
