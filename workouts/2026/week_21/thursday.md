# Thursday (21/05/2026)

**Setup**
- Equipment: Run lane, wall ball target, open floor
- Timing: 60 min hard cap
- Stimulus: Repeatable 5-minute intervals with a run, then unbroken wall balls and push-ups. Athletes should use the rest to recover enough to repeat the same pace six times.
- Scaling A: Shorter run or shuttle option if needed. Wall balls at 8/4 kg or reduce reps to 15. Push-ups to hand-release push-ups or incline push-ups.
- Scaling B: 200m shuttle run equivalent. Wall balls to a lighter target or thrusters with a light dumbbell. Push-ups to elevated hand-release push-ups.
- Logistics: Set one lane per athlete before class. Make sure the wall-ball target and push-up space are paired so the turn-around stays under 15 seconds.
- Top Cues: "Run first, then breathe." "Own the wall-ball set." "Keep the push-ups smooth."

## Warmup (6 min)
- 6:00 general warmup: easy jog or shuttle runs, then lunges, plank holds, and air squats
- Finish with 5 wall-ball-style squats and 5 push-ups

## Main Conditioning: Every 5:00 for 6 Sets
- 1 Lap run
- 20 Wall Ball Shots (10/6 kg) - unbroken
- 20 Push-Ups

**Target:** finish each set in about 2:30-3:15 and use the rest to reset.

## Cooldown (3 min)
1. 60-90s easy walk, nasal breathing
2. Hip flexor stretch: 45s/side
3. Chest opener on wall: 45s
