# Tuesday (19/05/2026)

**Setup**
- Equipment: Open run lane, machine or shuttle substitute, barbell, plates, box
- Timing: 60 min hard cap
- Stimulus: Long partner AMRAP with synchronized burpee mechanics, machine breathing, deadlift work, and a box-jump finish that rewards smooth pacing.
- Scaling A: Run to a shorter shuttle lane if needed. Down-ups to step-back burpees. Deadlifts at 100/70 kg. Box jumps to step-ups.
- Scaling B: Run to shuttle runs in place or 200m equivalent. Machine calories cut by 25-30%. Deadlifts at 80/55 kg. Box jumps to low step-ups or fast step-overs.
- Logistics: Split the room into run, machine, barbell, and box lanes before class. Partners should move with the same plan every round so transitions stay clean.
- Top Cues: "Stay together on the down-ups." "Own the deadlift set." "Land soft and keep moving."

## Warmup (6 min)
- 6:00 general warmup: easy jog or row, then lunges, hip hinges, and a few squat jumps
- Finish with 5 synchronized down-ups and 5 light deadlifts before starting

## Main Conditioning: Team of 2 - 35' AMRAP
- 1 Lap run together
- 20 Synchro Down-Ups
- 60 Calories on machine
- 20 Deadlifts (120/80 kg)
- 1 Lap run together
- 60 Box Jumps (60/50 cm)

Split the non-synchronized work any way between partners.

**Expected rounds:** 2-3 (fit: 3, ~10:00-12:00/round; average: 2-3, ~12:00-14:00/round)
