# Wednesday (10/06/2026)

**Setup**
- Equipment: Box (75/60 cm), rower, wall ball (10/6 kg), assault bike or machine, open run lane
- Timing: 60 min hard cap
- Stimulus: Eight 4-minute intervals across four mixed-modal stations, repeated twice. Each window pairs a fixed buy-in with a lap run, so athletes need crisp transitions and steady pacing rather than an early sprint.
- Scaling A: Box jumps to 60/50 cm or step-ups. Row and bike calories down to 12/10. Wall ball to 9/6 kg. Shorten the lap to 200 m or a shuttle equivalent if space is tight.
- Scaling B: Box step-ups without rebound. Row and bike to 10/8 calories. Wall ball to 6/4 kg. Reduce to 1 set of 4 stations if the class is running behind.
- Logistics: Stage all four stations in one flow lane: box → row → wall-ball target → bike → run return. Athletes stay in the same lane for both sets.
- Top Cues: "Land soft, stand tall." "Drive with the legs on the throw." "Run the turn, not the straightaway."

## Warmup (10 min)
- **R1 RAISE (2:00 easy):** Easy jog, bike, row, or shuttle runs
  - 6 x Bootstrappers
  - 8 x Air squats
- **R2 ACTIVATE (1:30 moderate):** Moderate row or bike build
  - 4 x Box jumps or step-ups bodyweight
  - 6 x Wall ball shots light
- **R3 POTENTIATE (1:00 hard):** 2 x 15s run acceleration with easy walk back
  - 3 x Box jumps at workout height or scale
  - 5 x Wall ball shots at workout load
- Rest 30s between rounds

## Post-Warmup Mobility (90s)
- Calf stretch on wall: 45s/side
- Hip flexor stretch: 45s/side

## Movement Prep (4 min)
- 1 short lap at workout pace
- 10 calories easy row
- 5 wall ball shots @ workout load
- 10 calories easy bike

## Main Conditioning: Every 4:00 for 2 Sets
Complete all 4 stations each set before repeating:

1. 15 Box Jumps (75/60 cm) + 1 Lap run
2. 15 Calories Row + 1 Lap run
3. 15 Wall Ball Shots (10/6 kg) + 1 Lap run
4. 15 Calories Assault Bike + 1 Lap run

**Target:** finish each 4-minute window with 20-40s left to reset. If athletes are missing that window, scale the box height or calorie targets first.

## Cooldown (3 min)
1. 60-90s easy walk, nasal breathing
2. Quad stretch on wall or box: 45s/side
3. Shoulder stretch on rig: 45s/side
