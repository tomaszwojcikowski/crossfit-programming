# Thursday (11/06/2026)

**Setup**
- Equipment: Barbell, plates, rack or open platform for overhead squat attempts
- Timing: 60 min hard cap
- Stimulus: Twenty minutes of progressive squat-snatch skill under the clock, then a short window to find a true 1RM overhead squat while positions are still fresh.
- Scaling A: Power snatch plus overhead squat if full squat snatch is not stable. Reduce load on the final EMOM block if catch positions break down.
- Scaling B: Snatch pull plus muscle snatch for the pause-below segment. Overhead squat 1RM attempt becomes a heavy triple at RPE 8 if mobility is the limiter.
- Logistics: One bar per athlete facing the same direction. Keep the rack clear for the 1RM attempt so athletes do not hunt for space after the EMOM.
- Top Cues: "Stay over the bar in the pause." "Punch up into the catch." "Set the feet before the overhead squat."

## Warmup (10 min)
- **R1 RAISE (2:00 easy):** Easy row, bike, or jog
  - 8 x Squat-to-stand
  - 8 x PVC pass-throughs
- **R2 ACTIVATE (1:30 moderate):** PVC rhythm work
  - 5 x PVC snatch-grip good mornings
  - 5 x PVC overhead squats
- **R3 POTENTIATE (1:00 hard):** 2 x PVC snatch balances
  - 3 x Empty-bar squat snatches
  - 3 x Empty-bar overhead squats
- Rest 30s between rounds

## Post-Warmup Mobility (90s)
- Lat stretch on rig: 45s/side
- Ankle rocks in squat: 45s

## Movement Prep (4 min)
- 2 x Squat snatch with 2-second pause below the knee, empty bar
- 1 x High hang squat snatch + 1 overhead squat, light load
- 1 x Hang squat snatch + 1 overhead squat, light load

## Strength / Skill: 20' EMOM
Build weight across the blocks. Add load only when positions stay crisp.

- **Minutes 1-5:** Squat Snatch with 2-second pause below the knee
- **Minutes 6-10:** High Hang Squat Snatch + 1 Overhead Squat
- **Minutes 11-15:** Hang Squat Snatch + 1 Overhead Squat
- **Minutes 16-20:** Squat Snatch + 1 Overhead Squat

**Target:** finish each minute in ~35-50s with 10-25s to change load or reset.

## Reset Window (3 min)
- Clear the platform, set safeties if available, and agree on a starting attempt for the overhead squat test.

## Main Conditioning: Find 1RM Overhead Squat

- Work up to a true 1-rep max overhead squat

**Time cap:** 8 min (expected: 5-8 min)

## Cooldown (3 min)
1. 60-90s easy walk, nasal breathing
2. Pigeon stretch: 45s/side
3. Wrist extension stretch: 30s/side
