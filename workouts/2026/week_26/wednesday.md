# Wednesday (24/06/2026)

**Setup**
- Equipment: Box, machine, open run lane
- Timing: 60 min hard cap
- Stimulus: Partner engine with simple movements and long repeatable sets. Teams should stay moving without redlining the first machine effort.
- Scaling A: Run to 200-300 m together. Down-ups from hands-elevated. Box jumps to lower box or fast step-ups. Machine calories to 70/60.
- Scaling B: Fast walk or shuttle run instead of the lap. Reduce synchronized down-ups to 14. Box step-ups for 30 reps total. Machine calories to 60/50.
- Logistics: One team per machine and box. Clarify the lap route before class and define that box jumps are "I go, you go" while the run and down-ups stay together.
- Top Cues: "Match pace on the run." "Chest and hips hit together on the down-up." "Step off the box and keep the turnover smooth."

## Warmup (10 min)
- **R1 RAISE (2:00 easy):** Easy jog or machine spin
  - 6 x Air squats
  - 4 x Down-ups
- **R2 ACTIVATE (1:30 moderate):** Build partner rhythm
  - 6 x Synchro air squats with partner
  - 6 x Box jump or step-up singles
  - 6/5 x Easy machine calories
- **R3 POTENTIATE (1:00 hard):** 20s brisk run effort
  - 4 x Synchronized down-ups
  - 4 x Box jumps at workout height
- Rest 30s between rounds

## Post-Warmup Mobility (75s)
- Ankle rocks on box: 30-45s/side
- Quad stretch: 30s/side

## Movement Prep (4 min)
- 60-90 m run together
- 6 x Synchronized down-ups
- 8 x Box jumps total, alternating singles
- 8/6 x Machine calories build to moderate pace

## Main Conditioning: Team of 2 — 26' AMRAP

- 1 Lap Run (together)
- 20 Synchronized Down-Ups
- 40 Box Jumps (I go, you go)
- 80/70 Machine Calories

**Expected rounds:** 2 rounds plus a partial third for most teams, with the best teams threatening 3 full rounds.

## Cooldown (3 min)
1. 60-90s easy walk, nasal breathing
2. Calf stretch on wall or box: 45s/side
3. Figure-4 glute stretch: 45s/side
