# Tuesday (23/06/2026)

**Setup**
- Equipment: Barbell, plates, rack or platform
- Timing: 60 min hard cap
- Stimulus: Overhead strength first, then barbell timing and footwork. This is a quality lifting day, not a conditioning race.
- Scaling A: Strict press percentages down 2-5% if lockout speed slows. Complex from the hang only if the floor pull is inconsistent.
- Scaling B: Seated or standing dumbbell strict press instead of barbell strict press. Complex to clean pull + hang power clean + push jerk with lighter load.
- Logistics: One bar per athlete for the full class. Keep plates close so load changes during the 90-second sets stay efficient.
- Top Cues: "Brace before the press." "Finish the first clean before rushing the hang clean." "Punch under the jerk, then stand."

## Warmup (10 min)
- **R1 RAISE (2:00 easy):** Easy row, bike, or jog
  - 8 x PVC pass-throughs
  - 8 x Tall-kneeling strict presses with PVC
- **R2 ACTIVATE (1:30 moderate):** Build overhead and pull sequence
  - 5 x Empty-bar strict presses
  - 3 x Clean pulls
  - 3 x Front squats
- **R3 POTENTIATE (1:00 hard):** 2 x Quick dip-drive bursts
  - 2 x Power cleans light
  - 1 x Hang clean + jerk light
- Rest 30s between rounds

## Post-Warmup Mobility (90s)
- Lat stretch on rig: 45s/side
- Wrist extension rocks: 45s

## Strength: Strict Press Ladder (11 min)
**Every 90s for 7 sets:**

- Set 1: 4 Strict Press @ 70% 1RM
- Set 2: 3 Strict Press @ 80%
- Set 3: 2 Strict Press @ 90%
- Set 4: 4 Strict Press @ 85%
- Set 5: 3 Strict Press @ 90%
- Set 6: 2 Strict Press @ 95%
- Set 7: 1 Strict Press @ 95%+

## Reset Window (2 min)
- Rebuild to the clean-complex starting load and review jerk footwork.

## Skill: Clean + Hang Clean + Jerk (15 min)
**Every 90s for 10 sets:**

- 1 Clean
- 1 Hang Clean
- 1 Jerk

Build load across the 10 sets while positions stay sharp.

**Target:** each set should finish in 20-35s, leaving time to recover and add load with intent.

## Cooldown (3 min)
1. 60-90s easy walk, nasal breathing
2. Forearm stretch: 30-45s/side
3. Triceps stretch: 30-45s/side
