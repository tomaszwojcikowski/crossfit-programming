# MONDAY: Front Squat 1RM + Team Bear Complex Engine

**Setup:**
*   **Time:** 60:00 hard cap | **Equip:** Barbell + plates, racks, machine (row/bike/ski)
*   **Flow:** Warmup 10:00 → Mobility 1:30 → Strength 12:00 → Reset 4:00 → Prep 3:00 → Metcon 21:00 → Cooldown/Debrief 8:30
*   **Stimulus:** Heavy single under control, then long team grind with barbell cycling + machine output.
*   **Standards:** Front squat below parallel. Full lockout overhead in each Bear rep. Team must complete full block before moving on.
*   **Scaling Priority:** Load → Volume → Complexity
*   **Scaling:**
	*   **A:** As written.
	*   **B:** Bear loads 50/30 → 60/35 → 70/40 → 80/45, machine 50 cal each block.
	*   **C:** 20/15/10/5 Barbell Complex (clean + front squat + push press) at 35/25 to 55/35, 40 cal each block.
*   **Top Cues:** "Drive the floor down." "Keep the bar close." "Breathe before each complex."
