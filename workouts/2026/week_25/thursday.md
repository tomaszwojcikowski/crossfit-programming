# Thursday (18/06/2026)

**Setup**
- Equipment: Dumbbells (22.5/15 kg pair), rower or assault bike, open run lane
- Timing: 60 min hard cap
- Stimulus: Six 5-minute windows alternate dumbbell snatch volume with machine calories, each paired with a lap run. Athletes need steady pacing and clean transitions, not an early sprint on round 1.
- Scaling A: Dual snatches to alternating snatches at same load. Machine to 10/8 calories. Shorten the lap to 200 m or a shuttle equivalent if space is tight.
- Scaling B: Dumbbells to 17.5/10 kg. Machine to 8/6 calories. Reduce to 4 sets if the class is running behind.
- Logistics: Set dumbbells, machine, and run lane in one flow: snatches → run → machine → run. Athletes stay in the same lane for all 6 sets.
- Top Cues: "Keep the dumbbells close on the turnover." "Run the turn, not the straightaway." "Drive with the legs on the machine."

## Warmup (10 min)
- **R1 RAISE (2:00 easy):** Easy jog, bike, row, or shuttle runs
  - 6 x Bootstrappers
  - 8 x Air squats
- **R2 ACTIVATE (1:30 moderate):** Build snatch rhythm
  - 6 x Alternating dumbbell snatches light
  - 10 calories easy row or bike
- **R3 POTENTIATE (1:00 hard):** 2 x 15s run acceleration with easy walk back
  - 4 x Dual dumbbell snatches light
  - 8 calories moderate on machine
- Rest 30s between rounds

## Post-Warmup Mobility (90s)
- Hamstring fold: 45s
- Hip flexor stretch: 45s/side

## Movement Prep (4 min)
- 1 short lap at workout pace
- 5 x Dual dumbbell snatches @ workout load
- 10 calories easy on machine
- 1 short lap at workout pace

## Main Conditioning: Every 5:00 for 6 Sets
Alternate A and B each set:

**A**
- 10 Dual Dumbbell Snatches (2 x 22.5/15 kg)
- 1 Lap run
- 10 Dual Dumbbell Snatches
- 1 Lap run

**B**
- 12 Calories on machine
- 1 Lap run
- 12 Calories on machine
- 1 Lap run

**Target:** finish each 5-minute window with 30-60s left to reset. Scale dumbbell load or machine calories first if athletes are missing the window.

## Cooldown (3 min)
1. 60-90s easy walk, nasal breathing
2. Calf stretch on wall: 45s/side
3. Forearm stretch: 30-45s/side
